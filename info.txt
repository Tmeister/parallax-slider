== Description ==

Use the Parallax slider to add a new feature slider look to your website. Slides display full size images and use thumbnails to help users find the information they want.

== Features ==

* Full Width Slider
* Thumbnail Navigation
* Pseudo 3D effect using 2 backgrounds
* Custom Post and Categories
* 7 options to play with
